VR Basketball

Play Game | Source Code on GitHub

Description

VR Basketball came about through a desire to explore VR gaming. In this game, one must shoot multiple basketballs into the hoops, with the furthest hoops garnering the most points. This was the first game I made that was Oculus Rift-compatible, and was made with the help of the Make Virtual Reality Games in Unity course on Udemy.

Technologies Used

Unity3d(v 5.4.1f), C#, GitHub

Date Made

October 2016
