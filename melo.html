Melo The Umang Boy

A Collaboration With Sari-Sari Storybooks

Description

Melo The Umang Boy is a 2D platformer video game collaboration with the publishing company, Sari-Sari Storybooks. Sari-sari Storybooks is a children's book company which aims to preserve cultural and linguistic diversity in the Philippines through publishing children's books in lesser-represented languages of the archipelago. While the outward purpose of the game is to reinforce Ivatan vocabulary words covered in the book, the hidden purpose of the game is to help make learning about Filipino culture fun. In order to make the game as accessible to as many children as possible, we made the decision to make this a web game.

Technologies Used

Unity3d(v.5.4.1f), AffinityPhoto (graphics editing), C#, GitHub

Date Made
