Javascript Tennis

Play Game | Source Code on GitHub

Description

JavaScript Tennis came about through a desire to explore HTML5's Canvas. I'd become accustomed to using powerful game englines like Unity3d, and I wanted to see what could be done with "just" JavaScript and the browser. This game was completed with the help of Chris DeLeon's Javascript Tennis game tutorial on Udemy.

Technologies Used

HTML5, Canvas, JavaScript, Sublime Text, GitHub

Date Made

October 2016
